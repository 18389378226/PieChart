# CustomView
显示数据百分比的环形图  
# Usage  
*新建Pieview  
```
PieView mPieView = new PieView(this);
mPieView.setStartAngle(0);
mPieView.setPieData(mPieDatas);
mPieView.setInvalidate();
linearLayout.addView(mPieView,params);
```
